## How to Build & Run

### Usar CMake (recomendado)

```
git clone https://github.com/Excalibur202/Miniprojeto3.git
cd Miniprojeto3

mkdir build
cd build

cmake ..
cmake --build .

./ProjectName
```

### Usar g++

```
g++ Miniprojeto3_Rui.cpp -o main
./main
```
